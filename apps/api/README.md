# API Workspace

This folder contains the NestJS backend, Prisma schema, Docker setup, and backend environment files.

## Common Commands

- `npm run start:dev --workspace @fonthabesha/api`
- `npm run build --workspace @fonthabesha/api`
- `npm run db:seed --workspace @fonthabesha/api`
- `npm run prisma:generate --workspace @fonthabesha/api`

## Local Files

- `src/` application code
- `prisma/` schema, migrations, and seed
- `docker-compose.yml` local services
- `.env.example` backend environment template
